# Nostra - Responsive Fashion Website

Nostra is a responsive fashion e-commerce website developed using HTML, Tailwind CSS, CSS, and JavaScript. The project demonstrates modern front-end development practices with a responsive layout, interactive components, and a clean user interface.

## Features

- Responsive design for mobile, tablet, and desktop devices
- Home page
- Collections page
- Contact page
- Product search functionality
- Contact form validation using JavaScript
- Image carousel
- Responsive navigation menu
- Clean and modern user interface

## Technologies Used

- HTML5
- Tailwind CSS
- CSS3
- JavaScript
